# 💬 ChatSphere – Real-Time Messaging Platform

ChatSphere is a production-style real-time messaging platform that enables secure one-to-one communication with instant messaging, image sharing, typing indicators, and read receipts. Built using modern web technologies, it leverages WebSockets, Redis caching, and RabbitMQ to deliver a fast, scalable, and reliable messaging experience.

---

## ✨ Features

- 🔐 Secure JWT Authentication
- 💬 Real-Time One-to-One Messaging
- 📱 OTP-Based User Authentication
- ⚡ Instant Messaging with Socket.IO
- ⌨️ Live Typing Indicators
- ✅ Message Read Receipts
- 🖼️ Image Sharing Support
- 🚀 Redis Caching for Faster Performance
- 📩 RabbitMQ-Based Asynchronous Messaging
- 🌐 Scalable REST APIs
- 📱 Responsive UI for Desktop & Mobile

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/your-username/ChatSphere.git
```

### Navigate to the Project

```bash
cd ChatSphere
```

### Install Dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env` file and add:

```env
MONGODB_URI=
JWT_SECRET=
REDIS_URL=
RABBITMQ_URL=
```

### Run the Development Server

```bash
npm run dev
```
